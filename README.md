# Synci

Synci is a simple application that allows you to synchronize your favorite sports events with your calendar.

## Frontend

### Requirements

- Node.js 20.17.0
- React with Vite

### Install
```bash
cd frontend && npm install
```

### Run the application

```bash
cd frontend && npm run dev
```

## Backend

### Requirements

- Java 21
- Gradle 8.10.1
- Spring Boot 3.3.3

### Run the application

```bash
./gradlew bootRun
```
